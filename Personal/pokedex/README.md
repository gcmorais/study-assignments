## 💻 Projeto

A ideia do projeto é construir uma Pokédex, uma enciclopédia digital onde fornece informações sobre todos os pokémons do mundo fictício.

## 👨‍💻 Tecnologias

    - HTML & CSS
    - Javascript
    - GIT & GITHUB
    - Node e NPM
    - Requisições em API (PokéAPI)
    - Vite
